# d12frosted.io

Personal site.

## Technologies Used

### Frontend
- [Next.js 15](https://nextjs.org/) - React framework with App Router and ISR
- [React 18](https://react.dev/) - UI library
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS v4](https://tailwindcss.com/) - Utility-first CSS framework
- [Tailwind Typography](https://tailwindcss.com/docs/typography-plugin) - Beautiful prose styles
- [Headless UI](https://headlessui.com/) - Unstyled, accessible UI components
- [Framer Motion](https://www.framer.com/motion/) - Animation library
- [Heroicons](https://heroicons.com/) - Icon set

### Content & Data
- [react-markdown](https://github.com/remarkjs/react-markdown) - Markdown rendering
- [react-syntax-highlighter](https://github.com/react-syntax-highlighter/react-syntax-highlighter) - Code syntax highlighting
- [remark-gfm](https://github.com/remarkjs/remark-gfm) - GitHub Flavoured Markdown
- [remark-math](https://github.com/remarkjs/remark-math) & [rehype-katex](https://github.com/remarkjs/remark-math/tree/main/packages/rehype-katex) - Math rendering
- [D3](https://d3js.org/) & [Observable Plot](https://observablehq.com/plot/) - Data visualisation

### Content Management
- [Emacs](https://www.gnu.org/software/emacs/) - Content authoring
- [Org Mode](https://orgmode.org/) - Markup format for blog posts
- [Vulpea](https://github.com/d12frosted/vulpea) - Note-taking library
- [org-roam](https://www.orgroam.com/) - Knowledge management

### Analytics & Deployment
- [Vercel Analytics](https://vercel.com/analytics) - Web analytics
- [Vercel](https://vercel.com/) - Hosting and deployment

## How to Use

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
